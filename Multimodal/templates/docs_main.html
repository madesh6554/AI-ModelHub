AI Model Hub Documentation

Overview

The AI Model Hub is a comprehensive platform that combines two powerful AI applications:

  • Gemini Analyzer: An advanced media analysis tool powered by Google's Gemini AI
  • Sign Language Translation: A real-time sign language detection and translation system

Technical Stack

  • Backend: FastAPI, Python
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • AI/ML: MediaPipe, OpenCV, Gemini AI
  • Database: SQLite (for user data and analysis results)

Setup and Installation

  1. Clone the repository
  2. Install dependencies: pip install -r requirements.txt
  3. Set up environment variables (if required)
  4. Run the application: python main.py

Project Structure

AI_ModelHub/
├── main.py              # FastAPI application
├── requirements.txt     # Python dependencies
├── static/             # Static files (CSS, JS, images)
├── templates/          # HTML templates
├── uploads/            # Uploaded files
└── README.md          # Project documentation

API Endpoints

  • GET /about/sign - Sign Language about page
